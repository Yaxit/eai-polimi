This folder contains different signal processing functions and libraries.
The contents of this folder are:

* gravity_filter.py : An IIR based gravity rotation and suppress function. This function is implementation of the Rodrigues formula.

* lib_features_extraction.py : python code for different feature extraction algorithms. This library include implementation for different features, such as nthMoment, magnitude, RMS, Skewness, Spread, Flatness, ZeroCrossEnvelop, Entropy and many more.

* feature_extraction.py : provides example calls and usage of different features in the "lib_features_extraction.py".

* readme.txt.
